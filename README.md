# 🚀 Landing Page - Em Desenvolvimento
## 📌 Sobre o Projeto

Este projeto é uma **landing page moderna em desenvolvimento**, criada com foco em design limpo, responsividade e boas práticas de UI/UX.

O objetivo é praticar e consolidar conhecimentos em desenvolvimento front-end, especialmente na criação de interfaces profissionais e responsivas.
